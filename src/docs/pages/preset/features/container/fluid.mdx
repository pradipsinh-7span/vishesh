# Fluid container

<Callout type="warning">
  Vishesh container is _**`tightly bounded with breakpoints`**_. so, it only
  generates container classes for breakpoints that's defined in the
  configurations.
</Callout>

> _please refer_ [breakpoints][breakpoints] <br /> `container width = 100% - (2 * spacing) ` &nbsp;<br />

### Reset Container

```css
/* Vishesh generates container-none class */
/* to reset the fluid container */

.container-none {
  width: initial;
  padding-left: unset;
  padding-right: unset;
  display: initial;
}
```

<Callout type="warning">
  Please consider [css specificity][specificity] when using `container-none`
</Callout>

<br />

### Basic usage

```js
class="container container-md"
```
