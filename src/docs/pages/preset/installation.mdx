# Installation

Install the Vishesh preset and set it up in your project.

## Pre-requisite

```shell
# Node.js (choose any)
- v14
- v16
- v18 (Recommended)

# Package manager (choose any)
- pnpm: v8 or higher (Recommended)
- npm: v7 or higher
- yarn: v1 or higher

# Tailwind CSS
- v3 or higher
```

<br />

## Install

<details open>
<summary>pnpm</summary>

```shell copy
pnpm add -D vishesh-preset
```

</details>

<details >
<summary>npm</summary>

```shell copy
npm i -D vishesh-preset
```

</details>

<details >
<summary>yarn</summary>

```shell copy
yarn add -D vishesh-preset
```

</details>

<br />

## Setup

1\. Set up [Tailwindcss][tailwindcss] &nbsp;&nbsp;(<small>_if not already done_</small>)

<Callout type="warning">
  <em>If you're using v2, [migrate to v3][migrate-to-v3]</em>
</Callout>

2\. Navigate to your Tailwind CSS configuration file. Then, identify which package to install based on how your config file is exported.

<Callout>
  <em>
    <abbr title="ECMAScript Module">_ESM_</abbr>
  </em>
  <em>: `export default`</em> <br />
  <em>
    <abbr title="CommonJS Module">*CJS*</abbr>
  </em>
  <em>: `module.exports`</em>
</Callout>

<br />

<details>
<summary>Setting up&nbsp;<abbr title="ECMAScript Module">_ESM_</abbr></summary>

> <em>You can extend or override the [configuration][tailwind-config].</em>

```diff filename="tailwind.config.js"
# /* your other imports */
+ import vishesh from "vishesh-preset";

# /* remove tailwindcss auto suggestion */
- /** @type {import('tailwindcss').Config} */

# /* add tailwindcss + vishesh-preset auto suggestion */
+ /** @type {import('vishesh-preset/types').VisheshPreset} */
  export default {
+   presets: [vishesh()],
    content: [____],
    theme: {
#     // ...
      extend: {
#     // ...
      }
    },
    plugins: [____]
  }
```

</details>

<details>
<summary>Setting up&nbsp;<abbr title="ECMAScript Module">_CJS_</abbr></summary>

> <em>You can extend or override the [configuration][tailwind-config].</em>

```diff filename="tailwind.config.cjs"
# /* your other imports */
+ const vishesh = require("vishesh-preset");

# /* remove tailwindcss auto suggestion */
- /** @type {import('tailwindcss').Config} */

# /* add tailwindcss + vishesh-preset auto suggestion */
+ /** @type {import('vishesh-preset/types').VisheshPreset} */
  module.exports = {
+   presets: [vishesh()],
    content: [____],
    theme: {
#     // ...
      extend: {
#       // ...
      }
    },
    plugins: [____]
  }
```

</details>
