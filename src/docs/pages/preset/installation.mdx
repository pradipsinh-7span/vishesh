## Setup

1\. Setup [Tailwindcss][tailwindcss] &nbsp;&nbsp;(<small>_if not already_</small>)

<Callout type="warning">
  <em>if running on v2, [migrate to v3][migrate-to-v3]</em>
</Callout>

2\. go to your tailwind config file. then, identify which package to install by looking at the way your config file were exported.

<Callout>
  <em>**ESM**: `export default`</em> <br />
  <em>**CJS**: `module.exports`</em>
</Callout>

<br />

### Setup ESM

> <em>you can extend or override [config][tailwind-config]</em>

```diff filename="tailwind.config.js"
# /* your imports */
+ import vishesh from "vishesh-preset";

# /* remove tailwindcss auto suggestion */
- /** @type {import('tailwindcss').Config} */
# /* add tailwindcss + vishesh-preset auto suggestion */
+ /** @type {import('vishesh-preset/types').VisheshPreset} */
  export default {
+   presets: [vishesh],
    content: [____],
    theme: {
#     // ...
      extend: {
#     // ...
      }
    },
    plugins: [____]
  }
```

<br />

### Setup CJS

> <em>you can extend or override [config][tailwind-config]</em>

```diff filename="tailwind.config.cjs"
# /* your imports */

# /* remove tailwindcss auto suggestion */
- /** @type {import('tailwindcss').Config} */
# /* add tailwindcss + vishesh-preset auto suggestion */
+ /** @type {import('vishesh-preset/types').VisheshPreset} */
  module.exports = {
+   presets: [require("vishesh-preset")],
    content: [____],
    theme: {
#     // ...
      extend: {
#       // ...
      }
    },
    plugins: [____]
  }
```
