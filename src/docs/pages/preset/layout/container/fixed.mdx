# Fixed container

<Callout type="info">
  <em>`max-width = breakpoint - ( 2 * spacing )`</em> <br />
  <em>For reference, please check [breakpoints][breakpoints]</em>
</Callout>

<br />

## Configuration

<details>
<summary>Max width</summary>

<Callout type="error">
  <em>Don't mix Auto and Manual classes</em>
</Callout>

<h4>Auto</h4>

> `container-fixed`

| Breakpoint | \<375px | ≥375px | ≥640px | ≥768px | ≥1024px | ≥1280  | ≥1440px | ≥1920px |
| :--------- | :------ | :----- | :----- | :----- | :------ | :----- | :------ | :------ |
| spacing    | 0       | 16     | 28     | 40     | 64      | 80     | 128     | 240     |
| max-width  | 100%    | 343px  | 584px  | 688px  | 896px   | 1120px | 1184px  | 1440px  |

<h4>Manual</h4>

1\. Base class `container` <small>(mandatory)</small> <br/>
2\. Max width class `container-x` <small>(where `x` is the breakpoint key)</small>

| container-maxw | 240     | 100%    | 100%   | 100%   | 100%   | 100%    | 100%    | 100%    | 1440px  |

</details>

<details>
<summary>Center</summary>

```js
container: {
  // ...
  center: true | false,
  // ...
}
```

<h4>true</h4>

- Horizontally align the fixed container to the center.
- This is set by default.

<h4>false</h4>

- Below classes are available to manually set alignment.

| Class            | Description                               |
| :--------------- | :---------------------------------------- |
| container-left   | Aligns the fixed container to the left.   |
| container-center | Centers the fixed container horizontally. |
| container-right  | Aligns the fixed container to the right.  |

</details>

<details>
<summary>Inner Padding</summary>

> Apply to all breakpoints

```js filename="apply to all breakpoints"
container: {
  // ...
  innerPadding: number | string,
  // ...
}
```

> Apply to specific breakpoint

```js filename="apply to specific breakpoint"
container: {
  // ...
  innerPadding: {
    "breakpoint key": number | string,
    // ...
  },
  // ...
}
```

| Value  | Condition                                                                                    | Default |
| :----- | :------------------------------------------------------------------------------------------- | :------ |
| Number | Should be 0 or greater                                                                       |         |
| String | Valid CSS like "16px", "1rem", etc...                                                        |         |
| object | Applies to specific breakpoint <br /> - Key: breakpoint key <br /> - Value: number or string | Yes     |

</details>

## Base class

```css
.container {
  width: 100%;
  display: block;
  /* If center is true */
  margin-right: auto;
  margin-left: auto;
  /* If innerWidth.zero set to 16px */
  padding-left: 16px;
  padding-right: 16px;
}
```

<br />

## Reset class

```css
.container-none {
  width: initial;
  margin-left: unset;
  margin-right: unset;
  padding-left: unset;
  padding-right: unset;
  max-width: unset;
  display: initial;
}
```

<br />

## Basic usage

<Callout type="error">
  <em>Don't mix Auto and Manual classes</em>
</Callout>

<h3 class="no-nx">Auto</h3>

> <em>Automatically set max-width at each breakpoint.</em>

```js
class="container-fixed"
```

<h3 class="no-nx">Manual</h3>

> <em>Manually set max-width at a specific breakpoint.</em>

```js
class="container container-md"
```
