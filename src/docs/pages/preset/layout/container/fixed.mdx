# Fixed container

<Callout type="warning">
  Vishesh container is _**`tightly bounded with breakpoints`**_. so, it only
  generates container classes for breakpoints that's defined in the
  configurations.
</Callout>

> _please refer_ [breakpoints][breakpoints] <br /> `max width = breakpoint - (2 * spacing) ` &nbsp;<br />

<br />

### If center set to false

then the following classes are generated:

| Class            | Description                         |
| ---------------- | ----------------------------------- |
| container-left   | Aligns the container to the left.   |
| container-center | Centers the container horizontally. |
| container-right  | Aligns the container to the right.  |

<br />

### Reset Container

```css
/* Vishesh generates container-none class */
/* to reset the fixed container */

.container-none {
  width: initial;
  margin-left: unset;
  margin-right: unset;
  max-width: unset;
  display: initial;
}
```

<Callout type="warning">
  Please consider [css specificity][specificity] when using `container-none`
</Callout>

<br />

### Basic usage

```js
class="container container-md"
```
