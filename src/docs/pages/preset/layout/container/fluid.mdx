## Configuration

### Horizontal padding

#### Auto

> `container-fluid`

| breakpoint | \<375px     | ≥375px      | ≥640px      | ≥768px      | ≥1024px      | ≥1280        | ≥1440px      | ≥1920px      |
| :--------- | :---------- | :---------- | :---------- | :---------- | :----------- | :----------- | :----------- | :----------- |
| spacing    | 16          | 16          | 28          | 40          | 64           | 80           | 128          | 240          |
| width      | 100% - 32px | 100% - 32px | 100% - 56px | 100% - 80px | 100% - 128px | 100% - 160px | 100% - 256px | 100% - 480px |

<Callout type="error">
  <em>Don't mix Auto and Manual classes</em>
</Callout>

#### Manual

1\. Base class `container` <small>(mandatory)</small> <br/>
2\. padding class `container-x` <small>(where `x` is the breakpoint key)</small>

| Class          | Spacing | \<375px     | ≥375px      | ≥640px      | ≥768px      | ≥1024px      | ≥1280px      | ≥1440px      | ≥1920px      |

## Base class

```css
.container {
  width: 100%;
  display: block;
  /* If spacing.zero set to 16px */
  padding-left: 16px;
  padding-right: 16px;
}
```

<br />

## Reset class

```css
.container-none {
  width: initial;
  margin-left: unset;
  margin-right: unset;
  padding-left: unset;
  padding-right: unset;
  max-width: unset;
  display: initial;
}
```

<br />

## Basic usage

<Callout type="error">
  <em>Don't mix Auto and Manual classes</em>
</Callout>

### Auto

> <em>automatically set horizontal padding at each breakpoint</em>

```js
class="container-fluid"
```

### Manual

> <em>manually set horizontal padding at a specific breakpoint</em>

```js
class="container container-md"
```
