# Core Plugins

The Vishesh Preset is a collection of plugins similar to Tailwind CSS. It generates classes only for the enabled plugins, ensuring that you have precisely what you needâ€”no more and no less.

By default, all the core plugins are enabled. To disable any of them, you can use `visheshCorePlugins`.

```js filename="tailwind.config.js"
export default {
  // ...
  visheshCorePlugins: {
    breakpoints: true,
    container: true,
    // ...
  },
  // ...
};
```
