# Core Plugins

The Vishesh Preset is a collection of plugins similar to Tailwind CSS. It generates classes only for the enabled plugins, ensuring that you have precisely what you need — no more and no less.

<Callout type="info">
  <em>By default, all the core plugins are enabled.</em>
</Callout>

<br />

## Disable Core Plugins

To disable any of the core plugins, add them to the disable array as shown below.

```js filename="tailwind.config.js"
export default {
  // ...
  presets: [
    vishesh({
      disable: ["container"],
    }),
  ],
  // ...
};
```

<Callout type="warning">
  <em>If you disable any of the `Depends on`</em>
  <em> core plugins as [listed below](#list-of-plugins),</em>
  <em> The dependent core plugins will also be disabled.</em> <br />
  <em className="font-semibold not-italic"> For example,</em> <br />
  <em>If you disable `breakpoints`, then `container` will also be disabled.</em>
</Callout>
