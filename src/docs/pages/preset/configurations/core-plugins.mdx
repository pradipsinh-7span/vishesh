# Core Plugins

Vishesh Preset is a collection of plugins similar to Tailwind CSS. It only generates classes for the enabled plugins. This ensures that you have precisely what you need, no more and no less.

By default, all the core plugins are enabled. To disable any of the it, you can use `visheshCorePlugins`.

```js filename="tailwind.config.js"
export default {
  // ...
  visheshCorePlugins: {
    breakpoints: true,
    container: true,
    // ...
  },
  // ...
};
```
