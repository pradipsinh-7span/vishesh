## Setup

1. Setup [Vite project][vite] &nbsp;&nbsp;<small>(_if not already_)</small>
2. Setup [Vishesh tsconfig][tsconfig] &nbsp;&nbsp;<small>(_if not already_)</small>
3. Setup [Vishesh prettier][prettier] &nbsp;&nbsp;<small>(_if not already_)</small>
4. Create two new files at the root of your repository.

<Tabs items={[".eslintrc", ".eslintignore"]}>

<Tab>

```json filename=".eslintrc" copy

  "extends": ["vishesh"],
  "rules": {}
}
```

## Basic scripts

> <em>Add scripts to lint your code</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "lint": "eslint . --ext .ts",
+   "fix:lint": "eslint . --ext .ts --fix"
  },
# // ...
}
```
