## Setup

1. Setup [Vishesh tsconfig-react][tsconfig-react] &nbsp;&nbsp;<small>(_if not already_)</small>
2. Setup [Vishesh prettier-react][prettier-react] &nbsp;&nbsp;<small>(_if not already_)</small>
3. Setup [Tailwindcss][tailwindcss] &nbsp;&nbsp;<small>(_if not already_)</small>
4. Create two new files at the root of your repository.

<Tabs items={[".eslintrc", ".eslintignore"]}>

<Tab>

```json filename=".eslintrc" copy

  "extends": ["vishesh-react"],
  "rules": {}
}
```

## Basic scripts

> <em>Add scripts to lint your code</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "lint": "eslint . --ext .ts,.tsx",
+   "fix:lint": "eslint . --ext .ts,.tsx --fix"
  },
# // ...
}
```
