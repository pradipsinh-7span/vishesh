> **<kbd>Typescript</kbd> + <kbd>React.js</kbd> + <kbd>Vite</kbd>**

## Prerequisites

```shell
# Node.js  (choose any)
- v14
- v16
- v18  (Recommended)

# Package manager  (choose any)
- pnpm: v8 or higher  (Recommended)
- npm: v7 or higher
- yarn: v1 or higher

# Tools
- React.js: v17 or higher
- Tailwindcss: v3 or higher
- Vite: v3 or higher
- Typescript: v4 or higher

# VS code extension
- dbaeumer.vscode-eslint
```

## Setup

1. Set up [Vishesh tsconfig-react][tsconfig-react] &nbsp;&nbsp;<small>(_if not already done_)</small>
2. Set up [Vishesh prettier-react][prettier-react] &nbsp;&nbsp;<small>(_if not already done_)</small>
3. Set up [Tailwindcss][tailwindcss] &nbsp;&nbsp;<small>(_if not already done_)</small>
4. Create two new files at the root of your repository.

<details open>
<summary>Config file</summary>

```json filename=".eslintrc" copy
{
  "extends": ["vishesh-react"],
  "rules": {}
}
```

<Callout type="info">
  <em>You can override [rules] and [other configs][configure-eslint].</em>
</Callout>

</details>

<details >
<summary>Ignore file</summary>

```js filename=".eslintignore" copy
.git
.husky
.next
dist
dist-ssr
node_modules
tsconfig.tsbuildinfo
types
package.json
*-lock.*
*.config.*
*.json
```

<Callout type="info">
  <em>_[Reference][eslint-ignore] for ignore files_.</em>
</Callout>

</details>

## Basic scripts

> <em>Add scripts to lint your code.</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "lint": "eslint . --ext .ts,.tsx",
+   "fix:lint": "eslint . --ext .ts,.tsx --fix"
  },
# // ...
}
```
