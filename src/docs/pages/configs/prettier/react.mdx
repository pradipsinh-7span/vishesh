## Setup

1. Setup [Vishesh tsconfig-react][tsconfig-react] &nbsp;&nbsp;<small>(_if not already_)</small>
2. Setup [Vishesh eslint-react][eslint-react] &nbsp;&nbsp;<small>(_if not already_)</small>
3. Setup [Tailwindcss][tailwindcss] &nbsp;&nbsp;<small>(_if not already_)</small>
4. Create two new files at the root of your repository.

<Tabs items={[".prettierrc", ".prettierignore"]}>

<Tab>

```json filename=".prettierrc" copy

  "plugins": ["vishesh-prettier-react"]
```

## Basic scripts

> <em>Add scripts to check formatting</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "check:formatting": "prettier --check \"**/*.{ts,tsx,html,css,json}\"",
+   "fix:formatting": "prettier --write \"**/*.{ts,tsx,html,css,json}\""
  },
# // ...
}
```
