# Prettier React.js config

> **<kbd>React.js</kbd> + <kbd>prettier-plugin-tailwindcss</kbd> + <kbd>clsx</kbd>**

## Prerequisites

```shell
# Node.js  (choose any)
- v14
- v16
- v18  (Recommended)

# Package manager  (choose any)
- pnpm: v8 or higher  (Recommended)
- npm: v7 or higher
- yarn: v1 or higher

# Tools
- Vite: v3 or higher
- Typescript: v4 or higher
- React.js: v17 or higher
- Tailwindcss: v3 or higher

# VS code extension
- dbaeumer.vscode-eslint
```

## Setup

1. Set up [Vishesh tsconfig-react][tsconfig-react] &nbsp;&nbsp;<small>(_if not already done_)</small>
2. Set up [Vishesh eslint-react][eslint-react] &nbsp;&nbsp;<small>(_if not already done_)</small>
3. Set up [Tailwindcss][tailwindcss] &nbsp;&nbsp;<small>(_if not already done_)</small>
4. Create two new files at the root of your repository.

<details open>
<summary>Config file</summary>

```json filename=".prettierrc" copy
{
  "plugins": ["vishesh-prettier-react"]
}
```

<Callout type="info">
  <em>You can override [options][prettier-opts].</em>
</Callout>

</details>

<details >
<summary>Ignore file</summary>

```js filename=".prettierignore" copy
.git
.next
dist
dist-ssr
node_modules
tsconfig.tsbuildinfo
types
*-lock.*
```

<Callout type="info">
  <em>_[Reference][prettier-ignore] for ignore files_.</em>
</Callout>

</details>

## Basic scripts

> <em>Add scripts to check formatting.</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "check:formatting": "prettier --check \"**/*.{ts,tsx,html,css,json}\"",
+   "fix:formatting": "prettier --write \"**/*.{ts,tsx,html,css,json}\""
  },
# // ...
}
```
