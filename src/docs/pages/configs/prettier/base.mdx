## Setup

1. Setup [Vite project][vite] &nbsp;&nbsp;<small>(_if not already_)</small>
2. Setup [Vishesh eslint][eslint] &nbsp;&nbsp;<small>(_if not already_)</small>
3. Setup [Vishesh tsconfig][tsconfig] &nbsp;&nbsp;<small>(_if not already_)</small>
4. Create two new files at the root of your repository.

<Tabs items={[".prettierrc", ".prettierignore"]}>

<Tab>

```json filename=".prettierrc" copy
{
  "plugins": ["vishesh-prettier"]
}
```

<Callout type="info">
  <em>you can override [options][prettier-opts]</em>
</Callout>

</Tab>

<Tab>

```js filename=".prettierignore" copy
types
*-lock.*
```

<Callout type="info">_[Reference][prettier-ignore ] for ignore files_</Callout>

## Basic scripts

> <em>Add scripts to check formatting</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "check:formatting": "prettier --check \"**/*.{js,ts,html,css,scss,json}\"",
+   "fix:formatting": "prettier --write \"**/*.{js,ts,html,css,scss,json}\""
  },
# // ...
}
```
