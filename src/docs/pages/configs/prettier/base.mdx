# Prettier base config

## Prerequisites

```shell
# Node.js  (choose any)
- v14
- v16
- v18  (Recommended)

# Package manager  (choose any)
- pnpm: v8 or higher  (Recommended)
- npm: v7 or higher
- yarn: v1 or higher

# VS code extension
- esbenp.prettier-vscode
```

<br />

## Install

<details open>
<summary>pnpm</summary>

    ```shell copy
    pnpm dlx install-peerdeps vishesh-prettier -P -D
    ```

</details>

<details >
<summary>npm</summary>

    ```shell copy

    ```

</details>

<details >
<summary>yarn</summary>

    yarn dlx install-peerdeps vishesh-prettier -Y -D

## Setup

1. Set up [Vite project][vite] &nbsp;&nbsp;<small>(_if not already done_)</small>
2. Set up [Vishesh eslint][eslint] &nbsp;&nbsp;<small>(_if not already done_)</small>
3. Set up [Vishesh tsconfig][tsconfig] &nbsp;&nbsp;<small>(_if not already done_)</small>
4. Create two new files at the root of your repository.

<details open>
<summary>Config file</summary>

```json filename=".prettierrc" copy
{
  "plugins": ["vishesh-prettier"]
}
```

<Callout type="info">
  <em>You can override [options][prettier-opts].</em>
</Callout>

</details>

<details >
<summary>Ignore file</summary>

```js filename=".prettierignore" copy
.git
.next
dist
dist-ssr
node_modules
tsconfig.tsbuildinfo
types
*-lock.*
```

<Callout type="info">
  <em>_[Reference][prettier-ignore ] for ignore files_.</em>
</Callout>

</details>

## Basic scripts

> <em>Add scripts to check formatting.</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "check:formatting": "prettier --check \"**/*.{js,ts,html,css,scss,json}\"",
+   "fix:formatting": "prettier --write \"**/*.{js,ts,html,css,scss,json}\""
  },
# // ...
}
```
