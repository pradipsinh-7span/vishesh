# Typescript base config

## Prerequisites

```shell
# Node.js  (choose any)
- v14
- v16
- v18  (Recommended)

# Package manager  (choose any)
- pnpm: v8 or higher  (Recommended)
- npm: v7 or higher
- yarn: v1 or higher
```

<br />
<br />

## Install

<details open>
<summary>pnpm</summary>

    ```shell copy
    pnpm dlx install-peerdeps vishesh-tsconfig -P -D
    ```

</details>

<details >
<summary>npm</summary>

    ```shell copy
    npm exec install-peerdeps vishesh-tsconfig -- -D
    ```

</details>

<details >
<summary>yarn</summary>

    ```shell copy
    yarn dlx install-peerdeps vishesh-tsconfig -Y -D
    ```

</details>

<br />
<br />

## Setup

1. Set up [Vite project][vite] &nbsp;&nbsp;<small>(_if not already done_)</small>
2. Set up [Vishesh eslint][eslint] &nbsp;&nbsp;<small>(_if not already done_)</small>
3. Set up [Vishesh prettier][prettier] &nbsp;&nbsp;<small>(_if not already done_)</small>
4. Create a new files at the root of your repository.

```json filename="tsconfig.json" copy
{
  "extends": "vishesh-tsconfig",
  "compilerOptions": {}
}
```

<Callout type="info">
  <em>You can override [options][tsconfig-opts].</em>
</Callout>

<br />
<br />

## Basic scripts

> <em>Add script to check types.</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "check:ts": "tsc"
  },
# // ...
}
```
