## Setup

1. Setup [Vite project][vite] &nbsp;&nbsp;<small>(_if not already_)</small>
2. Setup [Vishesh eslint][eslint] &nbsp;&nbsp;<small>(_if not already_)</small>
3. Setup [Vishesh prettier][prettier] &nbsp;&nbsp;<small>(_if not already_)</small>
4. Create a new files at the root of your repository.

```json filename="tsconfig.json" copy
{
  "extends": "vishesh-tsconfig",
  "compilerOptions": {}
}
```

<Callout type="info">
  <em>you can override [options][tsconfig-opts]</em>
</Callout>

<br />
<br />

## Basic scripts

> <em>Add script to check types</em>

```diff filename="package.json"
{
# // ...
  "scripts": {
#   // ...
+   "check:ts": "tsc"
  },
# // ...
}
```
